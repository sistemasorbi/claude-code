## ¿Qué es Claude Code?

**Claude Code es una herramienta de línea de comandos potenciada por IA que actúa como tu programador en pareja, ayudándote a construir, depurar y mejorar código a través de conversación natural.**

En lugar de escribir manualmente cada línea de código, describes lo que quieres lograr, y Claude Code te ayuda a implementarlo. Puede leer tus archivos, hacer ediciones, ejecutar comandos, buscar en la web y mucho más - todo mientras explica lo que está haciendo.

**[Comienza con Claude Code](https://github.com/anthropics/claude-code)**

## Conceptos Clave para Principiantes

### ¿Qué es la "Programación en Pareja con IA"?

> **Programación en Pareja con IA** es trabajar junto a la IA para construir software. En lugar de escribir cada línea de código tú mismo, describes lo que quieres, y la IA ayuda a implementarlo. Es como tener un desarrollador experto sentado a tu lado, listo para ayudar con cualquier tarea.

### Entendiendo la CLI (Interfaz de Línea de Comandos)

Claude Code se ejecuta en tu terminal. ¡No te intimides! Te enseñaremos todo lo que necesitas saber.

### Cómo Funciona Claude Code

1. **Tú:** Describes lo que quieres construir o cambiar
2. **Claude Code:** Analiza tu solicitud y hace preguntas clarificadoras si es necesario
3. **Asistente de IA:** Usa herramientas para leer archivos, escribir código, ejecutar comandos
4. **Tú:** Revisas los cambios y proporcionas retroalimentación
5. **Repetir:** Itera hasta que tengas lo que quieres

### Herramientas Explicadas Simplemente

- **Read** → "Muéstrame este archivo"
- **Write** → "Crea este nuevo archivo"
- **Edit** → "Cambia esta parte específica"
- **Bash** → "Ejecuta este comando"
- **Task** → "Maneja esta tarea compleja"
- **Grep** → "Encuentra este texto en mi código"
